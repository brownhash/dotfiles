# dotfiles

My personal dotfiles

## File locations

1. **vscode_settings.json**
    1. MAC `$HOME/Library/Application\ Support/Code/User/settings.json`
    2. Linux `$HOME/.config/Code/User/settings.json`
    3. Windows `%APPDATA%\Code\User\settings.json`
2. **.zshrc**
    1. MAC `$HOME/.zshrc`
    2. Linux `$HOME/.zshrc`
